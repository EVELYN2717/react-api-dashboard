# 🚀 React API Dashboard

🔗 Live Demo: https://react-api-dashboard.vercel.app/
💻 GitHub Repo: https://github.com/EVELYN2717/react-api-dashboard

---

Aplicación web desarrollada en React que consume una API externa, implementando buenas prácticas de arquitectura frontend.

## ✨ Características

- 🔍 Búsqueda en tiempo real
- 📄 Paginación dinámica
- ⏳ Loading spinner
- ⚠️ Manejo de errores
- ♻️ Componentes reutilizables
- 🧠 Custom Hook (`useCharacters`) para manejo de lógica
- 🎨 Interfaz moderna y responsive

## 🛠️ Tecnologías

- React + Vite
- TypeScript
- CSS
- REST API

## 🧠 Arquitectura

El proyecto sigue buenas prácticas de desarrollo frontend:

- Separación de responsabilidades
- Lógica desacoplada mediante hooks personalizados
- Componentes reutilizables
- Manejo de estado con hooks

## 🌐 API

Se utiliza la API pública de Rick and Morty:
https://rickandmortyapi.com/
