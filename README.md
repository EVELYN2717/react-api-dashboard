# 🚀 React API Dashboard

Aplicación web desarrollada en React que consume la API de Rick and Morty para mostrar un listado de personajes con funcionalidades modernas de frontend.

## ✨ Características

- 🔍 Búsqueda en tiempo real
- 📄 Paginación dinámica
- ⏳ Loading spinner
- ⚠️ Manejo de errores
- ♻️ Componentes reutilizables
- 🧠 Custom Hook (`useCharacters`) para manejo de lógica
- 🎨 Interfaz moderna y responsive

## 🛠️ Tecnologías

- React + Vite
- TypeScript
- CSS
- REST API

## 🧠 Arquitectura

El proyecto sigue buenas prácticas de desarrollo frontend:

- Separación de responsabilidades
- Lógica desacoplada mediante hooks personalizados
- Componentes reutilizables
- Manejo de estado con hooks

## 🌐 API

Se utiliza la API pública de Rick and Morty:
https://rickandmortyapi.com/
